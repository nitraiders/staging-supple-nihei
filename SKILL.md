# SKILL: NISHIN / KoKoRoMi 静的サイト

## 実装方針

### 1. Cloudflare Pages 前提
* `public` を公開ディレクトリとし、HTML/CSS/JS だけで成立する静的サイトとして管理する。
* Express、Render、サーバー常駐処理を前提にしない。
* 問い合わせや注文機能は、実装済みでない限り送信・購入できるように見せない。

### 2. 表記の原則
* サイト主体は「株式会社二信興業」。
* 株式会社二信興業は、サプリメント・化粧品等の販売、健康・美容関連商品の企画、顧客サポート / 電話案内などの販売支援を行う企業として見せる。
* 「KoKoRoMi」は主力商品 / ブランド名として扱う。
* 未確定のメールアドレス、URL、販売条件は捏造せず「現在準備中」と表記する。

### 3. 健康食品ページの注意
* 薬機法上リスクのある断定表現を避ける。
* 「治る」「改善する」「効く」「予防する」などは使わない。
* 「日々の健康習慣を支える」「健やかな毎日を目指す」など穏当な表現にする。

### 4. デザイン
* 健康食品と法人サイトに合う、落ち着いたグリーン基調の信頼感あるUIにする。
* スマホではハンバーガーナビを優先し、横揺れや文字はみ出しを避ける。
* 広告バナー画像に強い表現が含まれる場合は大きく使わず、商品写真と本文で丁寧に説明する。
